vbox-adjuster contains 2 packages that allow you to make some adjustments in VirtualBox so that it works properly in LliureX
# cdc-mapper-vboxusers
Package to configure the necessary permissions so that users in the admin, teacher, and students groups can use VirtualBox
# llx-vbox-config
Package to configure some settings in modprobe:
 - Currently, KVM is disabled for VirtualBox to function correctly on Linux kernel 6.12.
